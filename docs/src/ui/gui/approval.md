# 审批面板

## 功能说明

管理需要人工审批的待审批请求。当工具需要用户确认才能执行时，请求进入审批队列等待批准。

## 核心功能

- 列出所有待审批请求
- 查看请求详情（工具参数、调用上下文）
- 批准 / 拒绝请求
- 查看历史审批记录

## 工作流程

1. Agent 发起需要审批的工具调用
2. 请求进入审批队列
3. 用户在 Approval 面板查看详情
4. 用户批准 → 工具继续执行
5. 用户拒绝 → 工具取消执行

## 相关配置

在 `~/.klaw/config.toml` 中配置：

```toml
[approval]
enabled = true          # 是否启用审批流程
require_approval_for = [] # 需要审批的工具列表
```
